# Adi's Aloha Miles 🏝️🏃

26-week marathon training plan for the Honolulu Marathon (Dec 13, 2026).
Built around a 5% weekly mileage progression, Achilles tendinopathy
management, and a weekly schedule combining running, cycling, strength
training, and yoga.

## Local dev

```
npm install
npm run dev
```

## Deploy

Connected to Vercel — pushes to `main` auto-deploy.
